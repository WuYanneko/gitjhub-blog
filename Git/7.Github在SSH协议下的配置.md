### 1.新生成一个key及在github上添加新ssh密钥

如果不想使用已有的key或没有密钥，可以新生成一组，比如：
其中，ed25519是一种数字签名算法
也可用rsa算法（非对称加密协议）

==注意后面跟的邮箱和账户名是在安装git后，在本地库中创建的==（不是github上的）

ssh-keygen -t ed25519 -C d1547451414@163.com
(-t为指定XXX算法来生成ssh密钥，-C为描述生成的ssh存放在哪个账户下)
